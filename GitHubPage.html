C++ Implementation of Maximal Clique Algorithms

Introduction

Graph theory is fundamental in computer science, with applications ranging from social network analysis to bioinformatics. One critical problem in graph theory is maximal clique enumeration—identifying all maximal cliques (complete subgraphs that cannot be extended by adding another vertex) within a given graph. Efficient enumeration of maximal cliques has practical implications in diverse fields, including community detection, network security, and pattern recognition. This project focuses on implementing and analyzing three state-of-the-art maximal clique enumeration algorithms, each derived from prominent research papers. The aim is to understand their computational efficiencies and compare their performances across various graph datasets.

Implemented Algorithms

  • Procedure CLIQUE from "The Worst-case Time Complexity for Generating All Maximal Cliques and Computational Experiments" (Page 31).
  • Procedure BronKerboschDegeneracy from "Listing All Maximal Cliques in Sparse Graphs in Near-Optimal Time" (Figure 4).
  • Procedure CLIQUE from "Arboricity and Subgraph Listing Algorithms" (Pages 220-222).
